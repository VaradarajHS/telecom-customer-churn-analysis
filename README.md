# 📊 Telecom Customer Churn Analysis

## 📌 Project Overview

This project presents an end-to-end **Telecom Customer Churn Analysis** using **MySQL**, **Python**, and **Tableau**. The objective is to identify the factors influencing customer churn, analyze customer behavior, and provide actionable insights that can help improve customer retention.

The project demonstrates the complete analytics workflow, including:

- Data Cleaning & Preprocessing
- SQL Data Analysis
- Exploratory Data Analysis (EDA)
- Interactive Dashboard Development
- Business Insights & Recommendations

# 💡 Key Business Insights

### Customer Churn

- Month-to-month contracts experienced the highest churn.
- Long-term contracts retained more customers.

### Internet Services

- Fiber Optic customers showed higher churn compared to other internet services.

### Customer Satisfaction

- Customers with lower satisfaction scores were more likely to churn.

### Payment Method

- Electronic Check customers showed higher churn.

### Revenue

- Existing customers contributed the majority of total revenue.

---

# 📌 Business Recommendations

- Encourage customers to move to long-term contracts.
- Improve customer support for Fiber Optic users.
- Identify dissatisfied customers early.
- Offer loyalty programs.
- Improve onboarding experience for new customers.
- Monitor customers with high churn scores.
